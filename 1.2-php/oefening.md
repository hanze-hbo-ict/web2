# Oefening

## Functies

Op internet is een veelvoud aan oefenmateriaal te vinden om te oefenen met 
functies in PHP. Hierbij spelen twee aspecten een rol. Aan de ene kant is 
het schrijven van functies een vaardigheid die taalonafhankelijk is, waar 
het gaat om het selecteren en toepassen van het juiste algoritme. Dit kan 
bijvoorbeeld geoefend worden op de site van
[w3resource](https://www.w3resource.com/php-exercises/php-function-exercises.php).

Aan de andere kant heeft PHP een aantal specifieke functionaliteiten en 
keywords waarmee je kan oefenen. De kata's over PHP-functies op
[CodeWars](https://www.codewars.com/collections/php-functions) gaan met name 
over dit aspect. Kijk hierbij naar de kata's over Default Arguments, Pass By 
Reference, Splat Operator, Type Declarations en Return Type Declarations. De 
overige kata's op deze pagina gaan over aspecten van PHP die (nog) niet 
besproken zijn.


## Object-georiënteerd programmeren

Ook om te leren werken met objecten en klassen zijn voldoende bronnen op 
internet. Ook hier kan je kijken op de website van
[w3resource](https://www.w3resource.com/php-exercises/oop/index.php) en bij 
de kata's over object-georiënteerd programmeren in PHP op
[CodeWars](https://www.codewars.com/collections/object-oriented-php). De 
laatste kata in deze reeks, Objects on the Fly, gaat over anonieme klassen; 
dit is niet besproken. De overige kata's gaan over onderwerpen die wel 
besproken zijn.
