# Eindopdracht

Het vak Webtechnologie II wordt afgesloten met een opdracht, die kort gezegd 
inhoudt dat je een eenvoudig webframework moet ontwikkelen en daarmee een 
webapplicatie moet bouwen. Op Brightspace is een opdrachtomschrijving
te vinden waarin staat aan welke eisen de applicatie en het framework moeten 
voldoen.

De opdracht moet in versiebeheer worden opgeslagen. Je gebruikt hiervoor
[GitHub](https://github.com). Je verkrijgt de repository waarin je moet 
werken via GitHub Classroom; een link hiervoor is te vinden bij de 
opdrachtomschrijving op Brightspace. Om je werk in te leveren maak je een 
[Git-*bundle*](https://git-scm.com/docs/git-bundle) met onderstaand commando;
dit commando creeërt een bestand `web2.bundle` dat je vervolgens inlevert via 
Brightspace. Je mag de opdracht in duo's maken, mits je hierbij pair 
programming toepast.

```shell
git bundle create web2.bundle --all
```

Let erop dat dit vak een behoorlijk tijdsinvestering vergt. Het is in het 
algemeen niet mogelijk om de opdracht aan het eind van het blok nog in één 
of twee weken af te ronden; ervaringen uit voorgaande jaren leren dat het 
echt nodig is om hier vanaf het begin mee bezig te zijn. In de rest van het 
materiaal wordt per hoofdstuk steeds aangegeven met welk onderdeel van de 
eindopdracht je bezig zou moeten zijn.
